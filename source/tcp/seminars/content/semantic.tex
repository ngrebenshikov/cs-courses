Даны следующие высказывания и по два состояния для каждого:
\begin{enumerate}
	\item NOT (m OR n)
		\begin{enumerate}
			\item m=T, n=F
			\item m=F, n=T
		\end{enumerate}
	\item (m OR n) $\Rightarrow$ p
		\begin{enumerate}
			\item m=T, n=F, p=T
			\item m=F, n=T, p=F
		\end{enumerate}
	\item (m = n) AND (p = q)
		\begin{enumerate}
			\item m=F, n=F, p=T, q=F
			\item m=T, n=F, p=T, q=F
		\end{enumerate}
	\item (m $\Rightarrow$ (n $\Rightarrow$ p)) $\Rightarrow$ q
		\begin{enumerate}
			\item m=F, n=F, p=F, q=F
			\item m=T, n=T, p=T, q=T
		\end{enumerate}
\end{enumerate}

Вычислите высказывания для обоих состояний.
