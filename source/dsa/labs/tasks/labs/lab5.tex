\begin{problem}{Случайный фильм}{Стандартный поток ввода}{Стандартный поток вывода}{3 минуты}{16 Мегабайт}

Кинокомпания ``АбаканФильм'' снимает высокобюджетный блокбастер о приключениях супер-героев в Африке. Каждый день съемок приносит
несколько часов цифрового видео, которое требует последующей компьютерной обработки. Студии компании находятся в Хакасии в Абакане. Поэтому снятый материал необходимо каким-то образом переправить. Временной график создания фильмя очень строгий, поэтому было принято решение пересылать видео каждый день. Для большей скорости передачи данных использовали
несколько каналов. Даже была специально разработана программа, которая случайным образом выбирала кадр из текущего материала, и передавала по одному из каналов следующие данные: ``номер кадра'' и ``сам кадр''. Таким образом повторялось пока не будет передан весь фрагмент фильма. На принимающей стороне на каждом канале приемник записывал кадры фильма на жесткий диск. В итоге на каждом из приемников получалась случайная последовательность кадров фильма, которые затем склеивались, и получалась общая случайная последовательность кадров.

Вам необходимо написать программу, которая из этой случайной последовательности пар ``номер кадра - данные'' соберет искомый фильм.

\InputFile

Во входном файле вам даны полученные пары ``номер кадра - данные''. В первой строке задано число $N$ ($0 < N \leq 10^7$). В последующих $N$ строках записаны через пробел целые числа $X,Y$ ($X,Y\ni[-10^9,10^9]$), где $X$ - номер кадра, а $Y$ - данные кадра.

\OutputFile

В выходной файл необходимо вывести поледовательность данных кадров отсортированную по номеру кадра последовательность данных кадра. Каждый кадр должен быть записан на отдельной строке.

\Example

\begin{example}
\exmp{
3
1 10
3 30
2 20
}{
10
20
30
}%
\end{example}

\end{problem}
