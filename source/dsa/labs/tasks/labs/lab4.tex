\textbf{Интерфейс для АТД ``Список''}
\begin{lstlisting}[emph={insert,getPosition, getAt, deleteAt, next, previous, end, first, clear, sort},emphstyle=\underbar]
interface List<T, P> {
	void add(T o);
	T getAt(P position);
	P next(P position);
	P previous(P position);
	P end();
	P first();
	void clear();
	void sort(Comparator<T> comparator);
}
\end{lstlisting}

\begin{problem}{Планетарный исследователь}{Стандартный поток ввода}{Стандартный поток вывода}{4 секунды}

Тридцать лет два месяца и пять дней Максим блуждал по затерянным галактикам на просторах вселенной в поисках удивительных планетарных систем. Но сегодня его труды увенчались успехом. Его взору открылась ``идеальная'' планетарная система обладающая поразительными свойствами. В центре данной системы находится огромная звезда, вокруг которой вращаются $N$ планет. Орбиты каждой из планет представляют собой окружности, центром которых является звезда. К тому же все окружности находятся на одной плоскости. 

Максим решил выяснить имеются ли в данной системе такие различные планеты, которые вращаются вокруг звезды по одной и той же орбите. Для этого он выяснил координаты планет на плоскости орбит относительно звезды в определенный момент времени. Вам же предлагается определить количество орбит и количество планет на каждой орбите.

\begin{center}
\includegraphics[width=0.5\textwidth]{labs/imgs/lab4_1.pdf}

Рис. Пример расположения планет

\InputFile

Во входном файле вам даны координаты планет относительно звезды (звезда имеет координаты $(0,0)$). В первой строке задано число $N$ ($0 < N \leq 10^5$). В последующих $N$ строках записаны через пробел целые числа X,Y ($X,Y\ni[-10^6,10^6]$) - координаты каждой планеты.

\OutputFile

В выходной файл необходимо вывести через пробел количество планет на каждой из орбит рассматриваемой планетарной системы. Количество планет необходимо выводить в порядке удаления орбиты от звезды.

\Example

\begin{example}
\exmp{

1 3
3 1
-5 -5
2 2
-2 2
-3 -1
}{
2 3 1
}%
\end{example}
