
\textbf{Анализ программы}

Проведите анализ времени выполнения программы для наихудшего случая:

\begin{enumerate}
	\item Найдите функцию времени выполнения программы в зависимости от объема данных - $T(n)$
	\item Определите функцию минимально необходимой памяти для работы программы в зависимости от объема данных - $M(n)$
\end{enumerate}

\textbf{Отчет по лабораторной работе}

Отчет по лабораторной работе должен включать:
\begin{enumerate}
	\item Пронумерованный листинг программы, где для каждой строки программы (если это возможно) описано время ее выполнения.
	\item Дальнейший анализ временной сложности программы, где для групп операторов (циклов, вызовов функций) вычислено время их выполнения. В итоге вы должны получить временную сложность всей программы.
	\item Анализ минимально необходимой памяти для работы программы.
\end{enumerate}
