\begin{problem}{Сумма двух чисел}{Стандартный поток ввода}{Стандартный поток вывода}{2 секунды}

Найдите сумму двух чисел.

\InputFile

В первой строке входного файла заданы через пробел два целлых числа из диапозона $[0,10^9]$.

\OutputFile

Выходной файл должен содержать одно число - сумму.

\Example

\begin{example}
\exmp{
5 2
}{
7
}%
\end{example}

\Solution
На языке Java:
\begin{lstlisting}
import java.util.Scanner;

public class solver {
	public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int a = sc.nextInt();
        int b = sc.nextInt();
        System.out.print(a+b);
	}
}
\end{lstlisting}

\end{problem}

\begin{problem}{Сумма больших чисел}{input.txt}{output.txt}{2 секунды}

Вам необходимо реализовать абстрактный тип BigNumber, позволяющий работать с большими целыми числами. Абстрактный тип может быть реализован либо с помощью класса, либо с помощью структуры данных и функций для работы с ней. BigNumber должен хранить 100-значное число, иметь возможность записи значения из строки (с отслеживанием корректности этой строки), получения значения числа в виде строки, прибавление значения другого большого числа. Ниже приведен интерфейс BigNumber для реализации на Java

\begin{lstlisting}
interface BigNumber {
	void set(String str);
	String toString();
	void add(BigNumber num);
}
\end{lstlisting}

Используя этот абстрактный тип, чтобы решить задачу по нахождению суммы чисел.

\emph{Примечание:} При реализации абстрактного типа BigNumber нельзя использовать уже готовые решения для работы с длинной арифметикой. Такие как, например, java.math.BigInteger.

\InputFile

Во входном файле записаны целые из диапозона $[0,10^{100}]$. Каждое на отдельной строке.

\OutputFile

Выходной файл должен содержать одно число - сумму. Если во входном файле встречаются неверно записанные числа, или сумма чисел превышает $10^{100}-1$, то выходной файл должен содержать сообщение "invalid string" и "arithmetics overflow", соответственно.

\Example

\begin{example}
\exmp{
5
1
1
}{
7}%
\end{example}

\end{problem}
