	\begin{itemize}
		\item Сложные $\Rightarrow$ посещение обязательно.
		\item Выполняются в аудиторные часы.
		\item Язык программирования: Java.

	\end{itemize}

\begin{slide}
	\textbf{Жадная раскраска графа}

	\begin{enumerate}
		\item Выбираем произвольную незакрашенную вершину и назначаем ей новый цвет.
		\item Просматриваем список не закрашенных вершин и для каждой определяем, 
		соединина ли она ребром с вершиной, уже закрашенной в новый цвет. 
		Если нет, то к этой вершине также применяется новый цвет.
		\item Повторяем с начала пока все вершины не будут закрашены.
	\end{enumerate}
\end{slide}
