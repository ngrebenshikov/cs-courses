\section{Графы: сильносвязные компоненты и остовные деревья}

\subsection{Темы для рефератов (вопросы)}
\begin{enumerate}
		\item Доказательство теоремы: Процедура $FindStrongComp(G)$ корректно вычисляет сильно связные компоненты ориентированного графа $G$.
		\item Доказательство теоремы: Пусть $G=(V,E)$ - связный неориентированный граф с действительной весовой функцией $w$, определенной на $E$. Пусть $A$ - подмножество $E$,
	которое входит в некоторое минимальное остовное дерево $G$, $(S,V-S)$ - разрез $G$, согласованный с $А$ по ребрам, а $(u,v)$ - легкое ребро, пересекающее разрез
	$(S, V-S)$. Тогда ребро $(u,v)$ является безопасным для $A$.
		\item Стурктуры данных для непересекающихся подмножеств:
			\begin{itemize}
				\item представление с помощью связных списков
				\item лес непересекающихся подмножеств
			\end{itemize}
		\item Алгоритм Барувки (Baruvka)
\end{enumerate}
