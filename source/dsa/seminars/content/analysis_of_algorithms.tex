\section{Построение и анализ алгоритмов}

\subsection{}
В состязаниях футбольной лиги участвуют шесть команд:
\begin{itemize}
	\item Соколы;
	\item Львы;
	\item Орлы;
	\item Бобры;
	\item Тигры;
	\item Скунсы.
\end{itemize}
Уже сыграны следующие матчи:
\begin{itemize}
	\item Соколы - Львы и Орлы;
	\item Львы - Бобры и Скунсы;
	\item Тигры - Орлы и Скунсы.
\end{itemize}
Каждая команда играет одну игру в неделю. Найдите расписание матчей, чтобы все команды сыграли по одному разу друг с другом в течение минимального количества недель.

\textit{Задание взято из книги ``Структуры данных и алгоритмы''.}

\subsection{Решение рекурсий}

\subsubsection{Основной метод}
\begin{enumerate}
	\item $T(n)=2T(n/2) + n^3$
	\item $T(n)=T(9n/10) + n$
	\item $T(n)=16T(n/4) + n^2$
	\item $T(n)=2T(n/4) + \sqrt{n}$
	\item $T(n)=T(\sqrt{n}) + 1$
\end{enumerate}

\subsubsection{Дерево рекурсии}
\begin{enumerate}
	\item $T(n)=T(n/3) + T(2n/3) + n$
	\item $T(n)=T(n - a) + T(a) + n, a \geq 1$
	\item $T(n)=T(\alpha n) + T((1 - \alpha)n) + n, 0 < \alpha < 1$
	\item $T(n)=T(n-1) + n$
\end{enumerate}
