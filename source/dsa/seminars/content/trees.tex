\section{Деревья}

\subsection{Темы для рефератов (вопросы)}
\begin{enumerate}
	\item Области применения деревьев в задачах программирования.
	\item Префиксное дерево (Бор, Луч, Нагруженное дерево, Trie).
\end{enumerate}

\subsection{}
Напишите программу обхода $n$-арного дерева:
\begin{enumerate}
	\item в прямом порядке;
	\item в обратном порядке.
\end{enumerate}

\subsection{}
Напишите программу обхода бинарного дерева:
\begin{enumerate}
	\item в прямом порядке;
	\item во внутреннем  порядке;
	\item в обратном порядке.
\end{enumerate}

\subsection{}
Для приведенных арифметических выражений постройте деревья а затем переведите эти выражения в префиксную и постфиксную формы.
\begin{enumerate}
	\item $((a+b)+c*(d+e)+f)*(g+h)$
	\item $((a+b+c)*(g+f) *(d+e)+f)$
	\item $((a/b)+(c*((d+e)+f))*(g+h))$
\end{enumerate}

\subsection{}
\textit{Из книги ``Алгоритмы: построение и анализ 10.4-3''}

Разработайте нерекурсивную процедуру, которая за время $O(n)$ выводит ключи всех $n$ узлов бинарного дерева. 
В качестве вспомогательной структуры данных воспользуйтесь стеком.

\subsection{*}
\textit{Из книги ``Алгоритмы: построение и анализ 10.4-5''}

Разработайте нерекурсивную процедуру, которая за время $O(n)$ выводит ключи всех $n$ узлов бинарного дерева. 
Объем дополнительной памяти (кроме той, что отводится под само дерево) не должен превышать некоторую константу. 
Кроме того, в процессе выполнения процедуры дерево (даже временно) не должно модифицироваться.
